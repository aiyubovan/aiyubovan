Works with Google Authenticator, Authy, 1Password, Apple Passwords and similar apps. Once enabled, the database refuses every change unless the session passed the 2FA check.

1. Scan this QR code with your authenticator app (or type the secret manually).
2. Enter the 6-digit code it shows to activate.

Supabase currently offers TOTP (authenticator app) and SMS as second factors — passkey/WebAuthn login for project users isn't available yet. The moment they ship it, this panel can add it.
